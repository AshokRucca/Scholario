# 🎓 Scholario — Scholarship Eligibility & Discovery Platform

![Scholario Banner](./assets/banner.png)

## 🌟 Overview

**Scholario** is a premium Scholarship Eligibility & Discovery Platform designed to help students find, evaluate, and track scholarship opportunities from Government, NGO, University, and Private organizations.

Built entirely using **Vanilla HTML, CSS, and JavaScript**, Scholario delivers a fast, lightweight, and responsive experience without relying on frameworks or heavy dependencies.

The platform combines modern SaaS-inspired UI design with practical scholarship management tools, helping students discover funding opportunities more efficiently.

## 💡 Recommended Development Setup

For the best development experience, use **VS Code Live Server**:

1. Install the Live Server Extension.
2. Right-click `index.html`.
3. Select **Open with Live Server**.

* Automatic browser refresh
* Better module support
* Easier debugging
